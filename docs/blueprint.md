# **App Name**: Naija Slang Hub

## Core Features:

- Slang Listing: Display a comprehensive list of Nigerian street slangs on the home page.
- Slang Detail: Show the meaning and example usage of each slang on a dedicated page.
- Search Functionality: Enable users to quickly search for specific slangs using a search bar.
- Social Sharing: Integrate share buttons for WhatsApp, Instagram, and X to allow users to easily share slangs.
- Slang Submission: Allow users to submit new slang entries to expand the app's database. This will be stored to the cloud
